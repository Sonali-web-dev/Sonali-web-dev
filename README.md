## 🚀 Featured Project

### [AI-cruiter](https://github.com/Sonali-web-dev/AI-cruiter) — AI-Powered Resume Analyzer & Interview Assistant

> A full-stack MERN application that reduces manual resume review effort by **80%+**
> through AI-driven analysis, ATS scoring, and personalized improvement suggestions.

**✨ Key Features:**
- 📄 **AI Resume Analysis** — Automated parsing & keyword extraction via Google Gemini API (LLM)
- 🎯 **ATS Compatibility Scoring** — Skill gap detection with targeted improvement recommendations
- 📝 **Dynamic PDF Export** — ATS-optimized resume generation using Puppeteer (under 3s/request)
- 🔐 **Secure Authentication** — JWT-based auth with Role-Based Access Control (RBAC)
- ⚡ **10+ RESTful APIs** — Built with Express.js following MVC architecture on MongoDB Atlas
